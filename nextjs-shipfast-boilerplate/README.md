## Introduction

海外的 Marc Lou，在不断的创建新网站的路上，([他的产品列表](https://marclou.com/))
发现有些事情自己做了一遍又一遍，
例如设置DNS记录，监听stripe回调，设置google oauth登录等，
于是他便开发了 shipfast 这款产品用于解决上述问题。
shipfast使用了nextjs,mailgun,nextauth,stripe,tailwind这些组件，
来保证一个产品从想法到实现只需要五分钟。
